# Rubik-s-Timer

Rubicks Timer is a web-based stopwatch built with HTML, CSS, and JavaScript, demonstrating DOM manipulation, event handling, and interval management. The timer includes start/pause and reset controls, updating the display in real-time and changing button icons and colors according to the timer state.
